Pintos

Overview

This project is a course project of CS 162, Operating Systems and System Programming, and it simulated the OS via C mainly based on 3 topics: 1) User Program, 2) Threading and 3) File Systems. To implement each functionality, we used lots of data structures and synchronization techniques. In the first part, we solved the issues with argument passing and implemented some basic syscalls about process control and naive file systems. In the second part, we implemented priority scheduling of multi-threading. Finally, we build a buffer cache structure and implemented the FFS structure extensible files and directory structure based on it.
